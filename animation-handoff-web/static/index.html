美术统筹表工坊

上传已定稿脚本 Excel，即可导出与「美术统筹表范例」版式一致的美术统筹表（.xlsx）。

✨ 使用指南

方式一 · 工坊

默认在浏览器内直连 OpenAI 兼容接口。布局为三列：左侧本说明，中间上为 API、下为统筹表头（选填），右侧上传已定稿脚本并导出。

工坊主要读取脚本单元格的文字描述。若统筹表需要内嵌配图，请用方式二在 Cursor 中配合技能导出配图列。

方式二 · Cursor

将 animation-handoff-xlsx 技能放到本机 ~/.cursor/skills/animation-handoff-xlsx/，确保含 SKILL.md 与 scripts/generate_art_coordination_xlsx.py。重启 Cursor 后，在对话中调用该 SKILL 并附上脚本即可导出统筹表。

🤖 浏览器调用 API

可选 Gemini、OpenAI，或 OpenAI 兼容地址（国内/自建、Claude 网关等）。海外官方接口常被浏览器 CORS 拦截；国内是否允许浏览器直连以各平台说明为准，失败请勾选下方「服务端调用模型」，或使用 Cursor 技能。

📝 统筹表表头 选填

会写入导出表顶栏；与脚本内冲突时以脚本为准。

高级选项 · 工作表与行数
