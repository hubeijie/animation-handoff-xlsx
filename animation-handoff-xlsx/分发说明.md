# animation-handoff 技能 · 分发给他人

## 对方要放哪

把整个文件夹 **`animation-handoff-xlsx`** 复制到本机：

`~/.cursor/skills/animation-handoff-xlsx/`

（没有 `skills` 目录就新建。Windows 一般为 `%USERPROFILE%\.cursor\skills\animation-handoff-xlsx\`。）

## 依赖

```bash
python3 -m pip install openpyxl pillow
```

## 怎么在 Cursor 里用

1. 把带图脚本放进当前项目（或给出可读路径）。  
2. 对话里说清：按 **美术统筹 / animation-handoff** 生成 `.xlsx`。  
3. Agent 会读 **`SKILL.md`**（技能名 **`animation-handoff`**），按其中流程导出画面列、组 JSON、跑 `scripts/generate_art_coordination_xlsx.py`。

也可在 **Cursor Rules** 里写一句：做美术统筹表时遵循 `~/.cursor/skills/animation-handoff-xlsx/SKILL.md`。

## 网页版说明

### 静态说明（只读）

技能内附带 **`web/index.html`**。在技能根目录执行 `python3 -m http.server 8765`，浏览器打开（须含 `http://`，**不要** `www.`）：

**http://127.0.0.1:8765/web/**

### 上传脚本 → 下载统筹表（WebApp）

目录 **`webapp/`**：Flask 服务，上传脚本 xlsx → 返回美术统筹表 xlsx。

```bash
cd ~/.cursor/skills/animation-handoff-xlsx/webapp
python3 -m pip install -r requirements.txt
python3 app.py
```

打开 **http://127.0.0.1:8765/** 。描述为自动初稿，精细润色建议仍用 Cursor + `animation-handoff`。

## 包里有什么

| 路径 | 作用 |
|------|------|
| `SKILL.md` | 主流程与填表规则（Cursor 识别技能） |
| `web/index.html` | 静态说明页（可选） |
| `webapp/app.py` | 上传脚本生成统筹表（可选） |
| `webapp/requirements.txt` | WebApp 依赖 |
| `schema.md` | JSON 字段说明 |
| `reference/README.md` | 列位与空表说明 |
| `scripts/export_xlsx_column_images.py` | 按「画面」列导出内嵌图 |
| `scripts/export_xlsx_media.py` | 全量导出媒体 |
| `scripts/generate_art_coordination_xlsx.py` | JSON → 美术统筹表 xlsx |

## 打包给别人

在本机 `~/.cursor/skills/` 下执行：

```bash
zip -r ~/Desktop/animation-handoff-xlsx_skill.zip animation-handoff-xlsx
```

把生成的 **`animation-handoff-xlsx_skill.zip`** 发给对方；对方解压后按上文放到 `~/.cursor/skills/` 即可。
