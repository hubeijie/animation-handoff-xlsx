🗒️ 美术统筹表工坊

你当前打开的是本仓库在 GitHub Pages 上的说明页（静态托管）。

为什么这里没有「上传 Excel → 导出统筹表」？
完整工坊需要在本机或云服务器上运行 Python（FastAPI），用来生成与范例版式一致的 .xlsx。GitHub Pages 不能运行这类后端，因此无法把带导出功能的整站单独放在 Pages 里。

在本机使用（推荐）

  1. 克隆你的仓库（仓库名以你 GitHub 上为准，下面仅为示例）。
  2. 进入目录后安装依赖并启动：
cd animation-handoff-web python3 -m pip install -r requirements.txt ./start.sh

浏览器打开提示的地址（一般为 http://127.0.0.1:8765）。

代码仓库

若本页由 fork 部署，请将链接改成你的仓库地址：
github.com/hubeijie/animation-handoff-web

开启本说明页：在 GitHub 仓库 Settings → Pages 中，Source 选 Deploy from a branch，Branch 选 main，文件夹选 /docs，保存后等待几分钟访问 https://<你的用户名>.github.io/<仓库名>/。
